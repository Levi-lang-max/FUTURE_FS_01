# 🚀 Task 1 - Personal Portfolio Web App

This is a responsive personal portfolio website built using React and modern frontend tools. It showcases profile information, projects, and contact details.

---

## ✨ Features

- Home / Profile section
- About section
- Projects showcase
- Contact section
- Smooth navigation
- Responsive design (mobile + desktop)
- Clean modern UI

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- TanStack Router
